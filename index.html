üêß Linux Fundamentals

ls -la

Meaning: List All. Folder ki saari files dikhata hai, including hidden files (jo dot `.` se shuru hoti hain) aur unki details.

Practice: Check karne ke liye ki kya `.git` folder maujood hai.

pwd

Meaning: Print Working Directory. Ye batata hai ki tum abhi exactly computer ke kis folder ke andar khade ho.

Practice: Jab kho jao toh rasta pata karne ke liye.

mkdir -p

Meaning: Make Directory (Parent). Naya folder banane ke liye. `-p` se folder ke andar folder ek saath ban jata hai.

Practice: `mkdir -p projects/website/assets`

chmod +x

Meaning: Change Mode Executable. Kisi script file ko chalne ki power (permission) dene ke liye.

Practice: Health bot script banane ke baad use kiya tha.

üîç Search & Fix

grep "txt" file

Meaning: Global Regex Print. Kisi file ke andar koi specific shabd (word) ya line dhoondne ke liye.

Practice: Logs mein "error" dhoondne ke liye best hai.

cat file.txt

Meaning: Concatenate. Kisi file ko bina khole uska pura content terminal par print karne ke liye.

Practice: Chhoti scripts ya config files padhne ke liye.

üêô Git & GitHub Flow

git status

Meaning: Sabse zaroori command. Ye batata hai ki kya badla hai (Red) aur kya save hone ko taiyar hai (Green).

Practice: Har kadam par ise chalao, galti se bachogi.

git add .

Meaning: Stage All. Jitni bhi files tumne badli hain, un sabko ek saath "Packing Area" mein daal deta hai.

Practice: Dot (.) lagana mat bhoolna, uska matlab hai "sab kuch".

git commit -m ""

Meaning: Save Snapshot. Pack kiye hue changes ko ek message ke saath permanent save kar deta hai.

Practice: Message hamesha clear likho, jaise "Fixed design error".

git push origin main

Meaning: Go Live. Apne computer ke saved changes ko GitHub ke cloud server par bhej deta hai.

Practice: Spelling yaad rakhna: m-a-i-n, 'mian' nahi! üòâ

ü§ñ DevOps Automation Concepts

YAML Workflows

Meaning: `.github/workflows/deploy.yml` file. Ye wo "Recipe" hai jo GitHub ke robot ko batati hai ki website kaise banani hai.

Learnt: Isme 'environment: github-pages' dalna zaroori hai.

Token Permissions

Meaning: Security Keys. GitHub Action ko chalne ke liye 'Contents' aur 'Workflows' dono ki Write permission chahiye hoti hai.

Learnt: Error 403 aaye toh naya token banao.
